# Movie Management System (HashMap)

A Java application demonstrating the use of HashMap for storing and retrieving movie information.

## Features

* Add Movies
* Search Movies
* Display Movie Information

## Concepts Used

* HashMap
* Collections Framework
* Key-Value Data Storage

## How to Run

```bash
javac MovieManagementHashMap.java
java MovieManagementHashMap
```

# Movie Management System (Polymorphism)

A Java application demonstrating polymorphism through method overloading and dynamic behavior.

## Features

* Movie Information Handling
* Method Overloading
* Reusable Operations

## Concepts Used

* Polymorphism
* Method Overloading
* Classes and Objects

## How to Run

```bash
javac MovieManagementSystem.java
java MovieManagementSystem
```

# Bank Management System (Abstraction)

A Java application demonstrating the concept of abstraction using abstract classes and methods.

## Features

* Banking Operations
* Abstract Class Implementation
* Method Overriding

## Concepts Used

* Abstraction
* Inheritance
* Method Overriding

## How to Run

```bash
javac BankManagementAbstraction.java
java BankManagementAbstraction
```

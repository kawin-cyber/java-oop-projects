# Movie Management System (Encapsulation)

A Java application demonstrating encapsulation using private variables and controlled access through methods.

## Features

* Movie Information Management
* Data Protection
* Getter and Setter Methods

## Concepts Used

* Encapsulation
* Access Modifiers
* Classes and Objects

## How to Run

```bash
javac MovieManagementSystem.java
java MovieManagementSystem
```

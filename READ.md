# Java OOP Projects

A collection of Java projects demonstrating core Object-Oriented Programming (OOP) concepts and Java Collections Framework.

## Topics Covered

* Classes and Objects
* Abstraction
* Inheritance
* Encapsulation
* Polymorphism
* Exception Handling
* HashMap
* LinkedList

## Projects

### Bank Management System

Demonstrates exception handling and banking operations.

### Bank Management System (Abstraction)

Demonstrates abstraction using abstract classes and methods.

### Car Management System

Demonstrates inheritance concepts.

### Movie Management System (Encapsulation)

Demonstrates data hiding using private variables and getter/setter methods.

### Movie Management System (Polymorphism)

Demonstrates method overloading and runtime polymorphism.

### Movie Management System (HashMap)

Demonstrates data storage and retrieval using HashMap.

### Movie Management System (LinkedList)

Demonstrates data management using LinkedList.

### Student Information System

Demonstrates classes, objects, constructors, and basic data management.

## Technologies Used

* Java
* Object-Oriented Programming
* Java Collections Framework
